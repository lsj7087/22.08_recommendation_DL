고객 구매 데이터 기반 예측 모델 생성 및 개인화 마케팅 제안

주제 : 구매 경험이 없는 상품을 추천하는 추천시스템 개발

 - 사용자 별 구매 경험이 있는 상품들에 대하여 구매 패턴을 바탕으로 자체 선호도 산출
 
 - 인공신경망 기반 추천시스템을 활용하여, 구매 경험이 없는 상품들의 선호도를 예측
 
 - 예측된 선호도만을 바탕으로 개인에게 맞춤 추천 서비스를 제공하는 프로모션 제시
